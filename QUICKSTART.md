# Быстрый старт

## Установка и запуск

1. **Установите зависимости:**

```bash
npm install
```

2. **Запустите игру в режиме разработки:**

```bash
npm run dev
```

3. **Откройте браузер:**
   Игра автоматически откроется на `http://localhost:3000`

## Как играть

### Основные действия

1. **Клик по мобу** - атака моба
2. **Клик в пустое место** - атака ближайшего моба
3. **Автоатака** - включить/выключить автоматическую атаку
4. **Улучшить урон** - купить улучшение за золото

### Контракты

1. Откройте панель контрактов (левый нижний угол)
2. Выберите контракт для активации
3. Контракт влияет на:
   - Шанс спавна определенного типа моба
   - Получаемый опыт
   - Получаемое золото

### Типы мобов

- **Слайм** (зеленый) - слабый, часто появляется
- **Гоблин** (серый) - средний
- **Орк** (коричневый) - сильный
- **Босс** (красный) - очень сильный, редко появляется

## Отладка

В консоли браузера доступна глобальная переменная `window.game`:

```javascript
// Добавить золото
window.game.resources.addGold(1000);

// Активировать контракт
window.game.contractSystem.activateContract("slime_contract");

// Узнать уровень игрока
window.game.player.level;

// Узнать количество мобов
window.game.mobs.length;
```

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`
